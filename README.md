# TERMASTER

Статический сайт-визитка для мастера внутреннего ремонта в Санкт-Петербурге.

## Как открыть

Откройте `index.html` в браузере. Сборка, Node.js и backend не нужны.

## Где менять данные

- Контакты и SEO-текст: `index.html`
- Категории работ, прайс и галерея: `script.js`
- Цвета, адаптивность и общий вид: `styles.css`
- Изображения: `assets/images/`
